#### 4-setup the environment:

###### Clone this project in your machine using :

`git clone https://github.com/xxanisxx/Mini.git`  
###### run the project using:
`docker-compose up -d`  
###### install all environment needs:
`docker-compose exec php-fpm bash`  
`composer install`  
`bin/console d:m:m`  
`bin/console d:f:l`
